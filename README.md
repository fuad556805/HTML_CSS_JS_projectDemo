# 📚 StudyHub - Personal Student Dashboard

StudyHub is a simple, responsive, and beginner-friendly personal student dashboard built using **HTML5, CSS3, and Vanilla JavaScript**. The project combines several useful student utilities into a single web application, helping users manage their daily academic activities efficiently.

---

## ✨ Features

* ✅ To-Do List
* 📝 Notes
* 🧮 Calculator
* ⏱️ Stopwatch
* 🌍 World Clock
* ⚖️ BMI Calculator
* 📏 Unit Converter
* 📅 Calendar
* 🎯 Guess Game
* 🏠 Home Page
* ℹ️ About Page
* ❓ Help Page

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/your-username/StudyHub.git
```

### Open the project

```bash
cd StudyHub
```

### Run the project

This project loads feature pages using JavaScript **fetch()**.

Because of browser security (**CORS**), **do not open `index.html` by double-clicking it**.

Instead, run the project using a local server.

### Recommended Method (VS Code)

1. Open the project folder in **Visual Studio Code**.
2. Install the **Live Server** extension.
3. Right-click **index.html**.
4. Select **Open with Live Server**.
